# 📅 Daily Planning System

> **Purpose:** Time-block every day. No vague "I'll study later." Every hour has a job.
> **Created:** 2026-04-11

---

## Why This Exists

Weekly plans say **what** to do. Daily plans say **when** to do it.

Without time-blocking, work expands to fill all available time, and personal learning gets pushed to "later" (which means never). This system fixes that.

---

## How It Works

```
┌─────────────────────────────────────────┐
│  Every morning (or night before):       │
│  1. Open today's daily plan template    │
│  2. Fill time blocks based on:          │
│     - Work schedule (fixed)             │
│     - Weekly plan tasks (from W## file) │
│     - Energy levels (hard stuff early)  │
│  3. At end of day: mark done/skipped    │
│  4. Log blockers + carry forward        │
└─────────────────────────────────────────┘
```

## 📊 Priority Levels

| Priority | Meaning | Rule |
|----------|---------|------|
| 🔴 P0 | Non-negotiable | If you skip this, the day is a failure |
| 🟡 P1 | Important | Do it unless something P0 ran over |
| 🟢 P2 | Nice to have | Bonus points if done |

---

## 📏 Rules

1. **Plan the night before or first thing in the morning** — no winging it
2. **DSA is always P0** — the one thing that compounds most. Zero skip days.
3. **Speaking is P0** — 15 min is nothing. No excuses.
4. **Work hours are sacred** — don't leak personal study into SAP time (or vice versa)
5. **End-of-day review is mandatory** — 5 min to check boxes and plan tomorrow
6. **If you miss a block, DON'T try to "make up" by sacrificing sleep** — just carry forward
7. **Track energy, not just time** — if you're dead by 8pm, swap the hard block to morning
