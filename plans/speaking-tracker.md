## 🧰 Frameworks (Reference)

**PREP** (meetings, standups, pitches):
```
Point   → "The issue is X"
Reason  → "Because Y"  
Example → "For instance, Z"
Point   → "So that's why X"
```

**3 Act** (talks, presentations):
```
Act 1: Hook    → Story, question, or surprising fact (30 sec)
Act 2: Content → 3 points maximum. THREE.
Act 3: Close   → Callback to the hook. Clear takeaway.
```

**Feynman Technique** (explaining technical topics):
```
1. Say it in simple words (no jargon)
2. Use an analogy from daily life
3. If you can't simplify it, you don't understand it yet
```
