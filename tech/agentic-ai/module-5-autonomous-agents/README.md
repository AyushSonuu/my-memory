## 30-Second Recall 🧠
> **Planning** = don't hard-code tool sequences, let the LLM write its own plan (in JSON or code) and execute step by step. **Code > JSON > Text** for plan format. **Multi-agent** = decompose into specialized agents (researcher, designer, writer) with different prompts + tools. Two main communication patterns: **Linear** (relay race) and **Hierarchical** (manager coordinates). Deeper hierarchies and All-to-all exist but are harder to control.
